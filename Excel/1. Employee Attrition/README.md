
# 📊 Employee Attrition Analytics - Power BI
## 📌 Project Overview
This project analyzes **employee attrition trends** using **Microsoft Excel** to identify key factors contributing to workforce turnover.
The dataset was cleaned and analyzed using **Pivot Tables, Excel formulas, and charts** to uncover patterns related to employee attrition.\
The goal of this project is to demonstrate how **Excel can be used as a powerful data analysis tool** to generate insights for HR decision-making.

## 🖥️ Dashboard Preview
![Dashboard](https://github.com/da-shwetavir/portfolio_projects/blob/main/Excel/1.%20Employee%20Attrition/visuals/Excel_Employee_Attrition_Dashboard.png)
#### 🎯 Key Business Questions
This dashboard answers important HR analytics questions:
- What is the **overall attrition rate**?
- Which **departments experience the highest attrition**?
- How does **attrition vary by age group and gender**?
- Which **job roles show lower job satisfaction**?
## 📊Major Findings
✔ **R&D department** has the highest attrition (~56%). \
✔ Employees **aged 25–34** show the highest attrition numbers.\
✔ Life Sciences and Medical fields contribute the most employees leaving the company.\
✔ Certain job roles show lower job satisfaction ratings, indicating possible retention risks.

## 📈 Excel Analysis Features
The analysis was performed using the following Excel tools:\
**📊 Pivot Tables** – Summarized attrition trends across departments, age groups, and education fields.\
**📉 Pivot Charts** – Created visualizations for attrition comparisons.\
**📋 Data Cleaning** – Organized and structured the dataset for analysis.\
**📐 Excel Formulas** – Calculated metrics such as attrition rate and employee counts.
