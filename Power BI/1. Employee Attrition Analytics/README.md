# 📊 Employee Attrition Analytics - Power BI
## 📌 Project Overview
This project analyzes **employee attrition trends** using **SQL** and **Power BI** to identify key factors contributing to workforce turnover.
The goal is to help HR teams understand which **departments, job roles, education level, status**-*(Married, Single, Divorced)* experience the highest attrition and support data-driven retention strategies.

The dataset was analyzed using **SQL queries** and visualized through an interactive **Power BI** dashboard.

#### 🎯 Key Business Questions
This dashboard answers important HR analytics questions:
- What is the **overall attrition rate**?
- Which **departments experience the highest attrition**?
- How does **attrition vary by age group and gender**?
- Which **job roles show lower job satisfaction**?
## 📊Major Findings
✔ **R&D department** has the highest attrition (~56%). \
✔ Employees **aged 25–34** show the highest attrition numbers.\
✔ Life Sciences and Medical fields contribute the most employees leaving the company.\
✔ Certain job roles show lower job satisfaction ratings, indicating possible retention risks.
